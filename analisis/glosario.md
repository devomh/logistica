# Glosario de Términos Logísticos

Este glosario define los conceptos clave utilizados por el sistema para modelar la gestión logística pública.

| Término | Definición |
| :--- | :--- |
| **SIAF** | Sistema Integrado de Administración Financiera. Es el sistema externo que rige el presupuesto nacional. |
| **Meta Presupuestal** | El objetivo específico al cual se vincula un gasto. Ninguna compra existe sin una meta. |
| **Rubro / Fuente** | El origen de los fondos (ej. Recursos Ordinarios, Recursos Directamente Recaudados). |
| **CMN** | Cuadro Multianual de Necesidades. La programación de 4 años de lo que la entidad necesita. |
| **PECOSA** | Pedido Comprobante de Salida. El documento legal que autoriza la salida física de bienes del almacén hacia el usuario. |
| **NEA** | Nota de Entrada al Almacén. Documento que formaliza el ingreso de bienes por motivos distintos a una compra (donaciones, transferencias). |
| **Clasificador de Gasto** | Código numérico que identifica la naturaleza del bien o servicio (ej. Papelería, Alimentos). |
| **Kardex** | Registro permanente de entradas y salidas de inventario, valorizado mediante Precio Promedio. |
| **Centro de Costo** | Unidad organizacional (oficina o departamento) que genera la necesidad y consume el recurso. |
| **ESNI** | Estrategia Sanitaria de Inmunizaciones. Se refiere a ítems específicos (vacunas) que tienen controles de lote más estrictos. |
